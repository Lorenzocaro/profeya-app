# 📚 ProfeYa App

## 📌 Descripción del proyecto

**ProfeYa** es una aplicación web pensada para conectar estudiantes con profesores particulares de distintas materias.

El objetivo principal del proyecto es facilitar la búsqueda de docentes según las necesidades del alumno, ofreciendo una plataforma simple, rápida y accesible.

### ❓ Problema que intenta resolver

Muchas veces los estudiantes tienen dificultades para encontrar profesores particulares confiables o disponibles para determinadas materias.
ProfeYa busca simplificar ese proceso mediante una plataforma donde los usuarios puedan visualizar perfiles de docentes y acceder fácilmente a información relevante.

### 👥 Usuarios principales

* Estudiantes que buscan apoyo académico.
* Profesores particulares que desean ofrecer sus servicios.


## 🏠 Landing principal

* Página principal informativa.
* Presentación visual del proyecto.
* Navegación hacia otras secciones.

## 🔐 Registro de profesores

* Formulario para que los docentes puedan registrarse.

## 🔑 Login de profesores

* Inicio de sesión para profesores registrados.

## 🖼️ Carrusel de imágenes

* Carrusel automático incorporado en la landing principal.
* Mejora visual y dinámica de la página.

## 📚 Buscar profesores

* Nueva sección para visualizar perfiles de docentes.
